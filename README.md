# Pokedex Internship Project

A simple Pokemon search app built with Python and the PokeAPI.

## Milestone 1

- Retrieve Pokemon data from PokeAPI
- Provide a simple UI to search by Pokemon name or ID
- Display search results and random Pokemon details
- Include unit tests for data retrieval functions

## Run

1. Install dependencies:

   ```bash
   python -m pip install -r requirements.txt
   ```

2. Start the app:

   ```bash
   python -m app.main
   ```

3. Run tests:

   ```bash
   python -m unittest discover
   ```
